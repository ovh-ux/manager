## Install

```sh
$ yarn add @ovh-ux/ng-ovh-stop-event
```

## Usage

```js
import angular from 'angular';
import ngOvhStopEvent from '@ovh-ux/ng-ovh-stop-event';

angular.module('myApp', [ngOvhStopEvent]);
```

```html
<ANY data-ovh-angular-stop-event></ANY>
```

## Test

```sh
$ yarn test
```
