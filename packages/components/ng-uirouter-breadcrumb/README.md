## Install

```sh
yarn add @ovh-ux/ng-uirouter-breadcrumb
```
## Usage

```js
import angular from 'angular';
import ngUirouterBreadcrumb from '@ovh-ux/ng-uirouter-breadcrumb';

angular
  .module('myApp', [
    ngUirouterBreadcrumb,
  ]);
```

## Test

```sh
yarn test
```
