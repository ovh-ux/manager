## Install

```sh
yarn add @ovh-ux/manager-overthebox
```

## Usage

```js
import 'angular';
import managerOverthebox from '@ovh-ux/manager-overthebox';

angular
  .module('myApp', [
    managerOverthebox,
  ]);
```
